# Container

Containers are the most basic layout element in Chakra. It's used to constrain
the maximum of the content it wraps.

<ComponentLinks
  github={{ url: "#" }}
  storybook={{ url: "#" }}
  npm={{ label: "@chakra-ui/layout", url: "#" }}
/>

<carbon-ad></carbon-ad>

By default, the container constrains the width of it's content to 60 characters
(`60ch`). You can customize the container sizes in `theme.sizes.container`

While containers can be nested, most layouts do not require a nested container.
