# Avatar Group

AvatarGroup is a wrapper around avatars designed to render a collection of
avatars. You will want to use avatar group when you want to stack avatars, or
have easy collapse states once a set number of avatars are used, including a
dropdown to show hidden avatars.

It utilizes React's `cloneElement` to pass common props to the Avatars.

## Usage

```js
import { AvatarGroup } from "@chakra-ui/core";
```

```jsx
<AvatarGroup size="md">
  <Avatar name="Dan Abrahmov" src="http://bit.ly/dan-abramov" />
  <Avatar name="Uchiha Itachi" src="http://bit.ly/code-beast" />
  <MoreAvatarIndicator label="+4" />
</AvatarGroup>
```

## Props

The Avatar component composes `Box` component so you can pass props for `Box`.

| Name      | Type                                       | Default | Description                   |
| --------- | ------------------------------------------ | ------- | ----------------------------- |
| `size`    | `2xs`, `xs`, `sm`, `md`, `lg`, `xl`, `2xl` | `md`    | The size of the avatar        |
| `spacing` | `string` or `number`                       | `false` | The space between each avatar |
