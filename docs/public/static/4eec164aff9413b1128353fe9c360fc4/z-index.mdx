# Fighting the `zIndex` war

Every UI has stackable layers and it's important to have control over how the
layers play together.

Some examples of the stackable layers are tooltips, modals, popovers, select
dropdowns or menus.

The issues that come up along the layers usage are z-index and visibility
handling, what layer goes on top.
