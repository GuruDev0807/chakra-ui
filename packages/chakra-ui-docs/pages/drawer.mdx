# Drawer

The Drawer component is a panel overlaying the screen on either the left or
right side. It is used to show more details about a certain object or person. A
SideSheet is often triggered by clicking a row in a table. The Drawer component
can appear on all side of the screens; `top`, `right`, `bottom`, `left` using
the `placement` prop.

## Use cases

- Showing a profile view of a user.
- Showing detailed information about a transaction.
- Showing configuration settings that donâ€˜t need to be accessible by a URL.

## Installation

```bash
import { Drawer } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  const [isOpen, setIsOpen] = React.useState(false);
  return (
    <>
      <Drawer
        isOpen={isOpen}
        size="sm"
        placement="left"
        onClose={() => setIsOpen(false)}
      >
        <DrawerCloseButton
          onClick={() => {
            setIsOpen(false);
            console.log("hello");
          }}
        />
        <DrawerHeader>Create your account</DrawerHeader>

        <DrawerBody>
          <Input placeholder="Type here..." />
        </DrawerBody>

        <DrawerFooter>
          <Button variant="outline" mr={3} onClick={() => setIsOpen(false)}>
            Cancel
          </Button>
          <Button color="blue" onClick={() => setIsOpen(false)}>
            Save
          </Button>
        </DrawerFooter>
      </Drawer>
      <Button onClick={() => setIsOpen(true)}>Open Drawer</Button>
    </>
  );
}
```

## Accessibility

When opening the Drawer, focus will be brought inside the Drawer by looking for
elements with `autofocus` first, `tabindex` second and `button` last.

When closing the Drawer, focus will be brought back to the element that was
focused before opening the Drawer. This is normally the button that triggered
the Drawer.

## Props

| Name              | Type                                                                    | Default | Description                                                             |
| ----------------- | ----------------------------------------------------------------------- | ------- | ----------------------------------------------------------------------- |
| `isOpen`          | `boolean`                                                               | false   | If `true`, the drawer is shown.                                         |
| `onClose`         | `function`                                                              |         | Callback invoked when user closes the drawer.                           |
| `size`            | `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`, `4xl`, `5xl`, `6xl`, `full` | `md`    | The width of the drawer.                                                |
| `children`        | `React.ReactNode`                                                       |         | The children of the `InputGroup` e.g `InputLeftAddon`, `Input`.         |
| `isFullHeight`    | `boolean`                                                               | `false` | If `true`, the drawer fills the height of the viewport by default.      |
| `initialFocusRef` | `React.Ref`                                                             |         | The ref to the initial element to receive focus when the drawer opens. |
| `hideOverlay`     | `boolean`                                                               | `false` | If `true`, show the drawer overlay by default.                          |
| `placement`       | `top`, `left`, `bottom`, `right`                                        | `left`  | Set the position of the drawer.                                         |
