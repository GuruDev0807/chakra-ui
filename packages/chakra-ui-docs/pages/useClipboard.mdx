# useClipboard

`useClipboard` is a custom hook to handle copying content to clipboard.

## Return value

The `useClipboard` hook returns an object with the following fields:

| Name        | Type       | Default | Description                              |
| ----------- | ---------- | ------- | ---------------------------------------- |
| `value`     | `string`   |         | The copied value.                        |
| `onCopy`    | `function` |         | Callback function to copy content.       |
| `hasCopied` | `boolean`  | `false` | If `true`, the content has been copied . |

## Installation

```bash
npm install @chakra-ui/core
# or
yarn add @chakra-ui/core
```

And then import the hook.

```bash
import { useClipboard } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  const [value, setValue] = React.useState("Hello world");
  const [editableValue, setEditableValue] = React.useState("");

  const { onCopy, hasCopied } = useClipboard(value);

  return (
    <>
      <Flex>
        <Input value={value} placeholder="Welcome" />
        <Button onClick={onCopy} ml={2}>
          {hasCopied ? "Copied" : "Copy"}
        </Button>
      </Flex>
      <Editable
        mt={2}
        fontSize="md"
        isPreviewFocusable
        placeholder="Paste here"
        onSubmit={value => alert(value)}
      >
        <EditablePreview />
        <EditableInput />
      </Editable>
    </>
  );
}
```
