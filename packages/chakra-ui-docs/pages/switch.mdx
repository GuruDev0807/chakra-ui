# Switch

The Switch component is used as an alternative for the [Checkbox](/checkbox)
component, that simulates on/off instead of checked/unchecked.

Switch is optionally controllable, to control the component, specify a `value`
prop and an `onChange` prop.

## Installation

```bash
npm install @chakra-ui/core
```

then import Switch

```js
import { Switch } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  return <Switch />;
}
```

### Controlled Switch

The Switch component passes on the original event through the `onChange`
handler. Use `e.target.checked` to get the latest value and update state
accordingly.

```jsx
function Example() {
  let [value, setValue] = React.useState("");

  let handleInputChange = e => {
    let switchValue = e.target.checked;
    setValue(switchValue);
  };

  return (
    <>
      <Switch
        size="sm"
        value={value}
        color="green"
        onChange={handleInputChange}
      />
    </>
  );
}
```

## Switch sizes

```jsx
function Example() {
  return (
    <List inline>
      <Switch size="sm" />
      <Switch size="md" />
      <Switch size="lg" />
    </List>
  );
}
```

## Switch background color

```jsx
function Example() {
  return (
    <List inline>
      <Switch color="red" />
      <Switch color="gray" size="lg" />
    </List>
  );
}
```

## Props

| Name              | Type                                      | Default | Description                                          |
| ----------------- | ----------------------------------------- | ------- | ---------------------------------------------------- |
| `size`            | `sm`, `md`, `lg`                          |         | The size of the switch.                              |
| `color`           | `string`                                  |         | The background color of the switch when checked.     |
| `name`            | `string`                                  |         | The input name of the switch when used in a form.    |
| `value`           | `string`, `boolean`                       |         | The value of the switch.                             |
| `children`        | `React.ReactNode`                         |         | The children of the switch.                          |
| `aria-label`      | `React.AriaAttributes["aria-label"]`      |         | The aria-label of the switch for accessibility.      |
| `aria-labelledby` | `React.AriaAttributes["aria-labelledby"]` |         | The aria-labelledby of the switch for accessibility. |
| `isChecked`       | `boolean`                                 | `false` | If `true`, set the switch to the checked state.      |
| `isDisabled`      | `boolean`                                 | `false` | If `true`, set the disabled to the invalid state.    |
| `isInvalid`       | `boolean`                                 | `false` | If `true`, set the switch to the invalid state.      |
