# Code
Code is a component used to display . It is composed from the [Box](/box) component with a font family of `Mono` 

## Installation

```bash
import { Code } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  return <Code>Hello world</Code>;
}
```

## Props

| Name       | Type              | Default | Description                         |
| ---------- | ----------------- | ------- | ----------------------------------- |
| `children` | `React.ReactNode` | `false` | The children of the code component. |
