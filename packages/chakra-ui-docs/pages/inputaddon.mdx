# Input Addon

The InputGroup is a wrapper component used for the composition of input
components e.g an input or select with addons, icons or other elements on either
side of it.

## Installation

```js
import { InputAddon } from "@chakra-ui/core";
```

## Usage

```jsx
<InputGroup size="md">
  <InputLeftAddon>+234</InputLeftAddon>
  <Input roundedLeft="0" placeholder="Welcome" />
</InputGroup>
```

## Composition

Chakra exports 3 components you can use to compose more form input styles:

- InputGroup: A wrapper with `display: flex` that has uses React's cloneElement
  to pass it's props to its children
- InputLeftAddon: Used to prepend any element to the input
- InputRightAddon: Used to append any element to the input

Let's say we need an add-on to the left of the input, here's how we do it:

```jsx
<InputGroup size="md">
  <InputLeftAddon>+234</InputLeftAddon>
  <Input roundedLeft="0" placeholder="Welcome" />
</InputGroup>
```

Similar patter for the right:

```jsx
<InputGroup size="md">
  <InputRightAddon>.com</InputRightAddon>
  <Input roundedRight="0" placeholder="Welcome" />
</InputGroup>
```

## Props

| Name     | Type              | Default | Description                                                     |
| -------- | ----------------- | ------- | --------------------------------------------------------------- |
| size     | `sm`, `md`, `lg`  | `md`    | The size of the input component composed with the `InputGroup`. |
| children | `React.ReactNode` |         | The children of the `InputGroup` e.g `InputLeftAddon`, `Input`. |
