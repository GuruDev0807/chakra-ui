`useColors` is a custom hook to resolve design tokens from the theme as actual
CSS colors.

## Import

```js
import { useColors } from "@chakra-ui/core"
```

## Return value

The `useColors` hook returns an array of resolved colors. For unknown colors, it
will return `undefined`.

## Usage

```jsx live=true
function Example() {
  const [teal100, red300] = useColors("teal.100", "red.300")

  return (
    <Box bg={`linear-gradient(${teal100}, ${red300})`}>
      <Text>wonderful gradients</Text>
    </Box>
  )
}
```
