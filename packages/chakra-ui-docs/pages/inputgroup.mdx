# Input Group
Easily extend form controls by adding text, buttons, or icon on either side of
textual inputs or selects, and custom file inputs. For example, you might use an
input group to build a visibility toggle for a password field.

## Installation

```bash
npm install @chakra-ui/core
```

## Usage

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftAddon>+234</InputLeftAddon>
      <Input roundedLeft="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

## Compostion

Use the `InputGroup` to compose form input components by placing an
[addon-on](/inputaddon), [Icon](/icon) or [Button](/button) on either side of an
[Input](/input). You may also place one on both sides of an input as well.

### With left addon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftAddon>+234</InputLeftAddon>
      <Input roundedLeft="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With right addon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputRightAddon>.com</InputRightAddon>
      <Input roundedRight="0" placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With left icon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputLeftElement>
        <Icon name="phone" size="1em" />
      </InputLeftElement>
      <Input placeholder="Welcome" />
    </InputGroup>
  );
}
```

### With right icon

```jsx
function Example() {
  return (
    <InputGroup size="md">
      <InputRightElement>
        <Icon name="check" size="1em" />
      </InputRightElement>
      <Input placeholder="Welcome" />
    </InputGroup>
  );
}
```

## Props

| Name       | Type              | Default | Description                                                     |
| ---------- | ----------------- | ------- | --------------------------------------------------------------- |
| `size`     | `sm`, `md`, `lg`  | `md`    | The size of the input component composed with the `InputGroup`. |
| `children` | `React.ReactNode` | `false` | The children of the `InputGroup` e.g `InputLeftAddon`, `Input`. |
