# Avatars

The Avatar component is used to represent users.

The `Avatar` package export 2 Components:

- `Avatar`: This displays the image that represents the user.
- `AvatarBadge`: In some products, you might need show a badge on the right
  corner of the avatar. For instance, an avatar can show if the user is online.

## Usage

```js
import { Avatar } from "@chakra-ui/core";
```

```jsx
<List isInline>
  <Avatar name="Dan Abrahmov" src="http://bit.ly/dan-abramov" />
  <Avatar name="Kola Tioluwani" src="http://bit.ly/tioluwani-kolawole" />
  <Avatar name="Kent Dodds" src="http://bit.ly/kent-c-dodds" />
  <Avatar name="Ryan Florence" src="http://bit.ly/ryan-florence" />
  <Avatar name="Prosper Otemuyiwa" src="http://bit.ly/prosper-baba" />
  <Avatar name="Christian Nwamba" src="http://bit.ly/code-beast" />
</List>
```

## Avatar Sizes

The Avatar component comes in 7 sizes

```jsx
<List isInline>
  <Avatar size="2xs" name="Dan Abrahmov" src="http://bit.ly/dan-abramov" />
  <Avatar
    size="xs"
    name="Kola Tioluwani"
    src="http://bit.ly/tioluwani-kolawole"
  />
  <Avatar size="sm" name="Kent Dodds" src="http://bit.ly/kent-c-dodds" />
  <Avatar size="md" name="Ryan Florence" src="http://bit.ly/ryan-florence" />
  <Avatar size="lg" name="Prosper Otemuyiwa" src="http://bit.ly/prosper-baba" />
  <Avatar size="xl" name="Christian Nwamba" src="http://bit.ly/code-beast" />
  <Avatar size="2xl" name="Segun Adebayo" src="http://bit.ly/sage-adebayo" />
</List>
```

## Avatar Fallbacks

If there was an error loading the image `src` of the avatar, we do two things:

- If there's a `name` prop, we use it to generate the initials and a random
  background color.
- If there's no `name` prop, we use a default avatar

```jsx
<List isInline>
  <Avatar name="Oshigaki Kisame" src="http://bit.ly/broken-link" />
  <Avatar name="Sasuke Uchiha" src="http://bit.ly/broken-link" />
  <Avatar src="http://bit.ly/broken-link" />
</List>
```

## Avatar with badge

```jsx
<Avatar>
  <AvatarBadge size="1.25em" bg="green.500" />
</Avatar>
```

> The `em` size of the `AvatarBadge` helps us size the circle relative to the
> avatar size

## Props

The Avatar component composes `Box` component so you can pass props for `Box`.

| Name         | Type                                       | Default | Description                                                                       |
| ------------ | ------------------------------------------ | ------- | --------------------------------------------------------------------------------- |
| `size`       | `2xs`, `xs`, `sm`, `md`, `lg`, `xl`, `2xl` | `md`    | The size of the avatar                                                            |
| `showBorder` | `boolean`                                  | `false` | If `true`, the `Avatar` will show a border around it. Best for a group of avatars |
| `name`       | `string`                                   |         | The name of the user in the avatar                                                |
| `src`        | `string`                                   |         | The image url of the `Avatar`                                                     |
| `children`   | `React.ReactNode`                          |         | The badge at the bottom right corner of the Avatar.                               |
