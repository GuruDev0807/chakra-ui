# Pin Input

The PinInput component is similar to the [Input](/input) component, but it is
optimized for entering sequences of digits. The most common application is for
entering single-use security codes. It is optimized for entering digits quickly.

## Import

```js
import { PinInput, PinInputField } from "@chakra-ui/core"
```

## Usage

Here's a basic usage example of the Input component:

```jsx
<PinInput>
  <PinInputField />
  <PinInputField />
  <PinInputField />
  <PinInputField />
</PinInput>
```

### Changing the size of the Input

There are three sizes of an Input:

- `sm` (`24px`)
- `md` (`32px`)
- `lg` (`40px`)

```jsx
<>
  <Box mb={4}>
    <PinInput size="lg">
      <PinInputField />
      <PinInputField />
      <PinInputField />
    </PinInput>
  </Box>
  <Box mb={4}>
    <PinInput size="md">
      <PinInputField />
      <PinInputField />
      <PinInputField />
    </PinInput>
  </Box>
  <Box>
    <PinInput size="sm">
      <PinInputField />
      <PinInputField />
      <PinInputField />
    </PinInput>
  </Box>
</>
```

### Adding a defaultValue

You can set the defaultValue of the PinInput if you wish:

```jsx
<PinInput defaultValue="234">
  <PinInputField />
  <PinInputField />
  <PinInputField />
</PinInput>
```

Or even a partial defaultValue:

```jsx
<PinInput defaultValue="23">
  <PinInputField />
  <PinInputField />
  <PinInputField />
</PinInput>
```

## Props

### PinInputField

PinInputField composes `Input` so you can pass all `Input` props.
