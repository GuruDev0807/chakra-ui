# CloseButton

CloseButton is essentially a button with a close icon. It is used to handle the
close functionality in feedback components: [Alert](/alert), [Toast](/toast), as
well as overlay components: [Drawer](/drawer) and [Modal](/modal).

## Installation

```bash
import { CloseButton } from "@chakra-ui/core";
```

## Usage

```jsx
function Example() {
  return <CloseButton />;
}
```
