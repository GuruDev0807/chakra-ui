# Heading

Headings are used for rendering headlines.

Heading composes `Box` so you can use all the style props and add responsive
styles as well. It renders an `<h2>` tag by default.

## import

```js
import { Heading } from "@chakra-ui/core";
```

## Usage

```jsx
<Heading>Heading component</Heading>
```

### Size

The level of headline you select should provide the appropriate hierarchy and
visual coherency within the interface.

> If you use the `size` prop, the `Heading` component will automatically
> decrease in size for smaller screens.

```jsx
<>
  <Heading size="2xl">Heading 1</Heading>
  <Heading size="xl">Heading 2</Heading>
  <Heading size="lg">Heading 3</Heading>
  <Heading size="md">Heading 4</Heading>
  <Heading size="sm">Heading 5</Heading>
  <Heading size="xs">Heading 6</Heading>
</>
```

### Override style

You can override the size of the Heading component by passing the `fontSize`
prop. No need to write `css` or `styled()`

```jsx
<Heading size="lg" fontSize="50px">
  I'm overriding this heading
</Heading>
```

## Composition

```jsx
function PaystackExample() {
  return (
    <>
      <Heading lineHeight="shorter" size="2xl" mb={4}>
        Modern online and offline payments for Africa
      </Heading>
      <Text fontSize="xl">
        Paystack helps businesses in Africa get paid by anyone, anywhere in the
        world
      </Text>
      <Button size="lg" variantColor="green" mt="24px">
        Create a free account
      </Button>
    </>
  );
}
```

## Props

| Name   | Type                                 | Default | Description              |
| ------ | ------------------------------------ | ------- | ------------------------ |
| `size` | `2xl`, `xl` , `lg`, `md`, `sm`, `xs` | `md`    | The size of the heading. |
