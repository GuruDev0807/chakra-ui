# CSSReset

CSSReset is a component used to apply low-level CSS rules to allow complete,
bug-free and consistent styling by Chakra.

<carbon-ad></carbon-ad>

## Import

```js
import { CSSReset } from "@chakra-ui/core";
```

## Usage

Include CSSReset as a child of ColorModeProvider in order to automatically style
tags like `<html>` according to the color mode.

```jsx live=false
<ThemeProvider>
  <ColorModeProvider>
    <CSSReset />
    <Component />
  </ColorModeProvider>
</ThemeProvider>
```

## Custom Configuration

By default CSSReset uses these values:

```jsx live=false
const defaultConfig = theme => ({
  light: {
    color: theme.colors.gray[800],
    bg: undefined,
    borderColor: theme.colors.gray[200],
    placeholderColor: theme.colors.gray[400],
  },
  dark: {
    color: theme.colors.whiteAlpha[900],
    bg: theme.colors.gray[800],
    borderColor: theme.colors.whiteAlpha[300],
    placeholderColor: theme.colors.whiteAlpha[400],
  },
});
```

To create a custom config, first define it and adjust colors as needed:

```jsx live=false
const customConfig = theme => ({
  light: {
    color: theme.colors.someCustomColor[400],
    bg: undefined,
    borderColor: theme.colors.gray[200],
    placeholderColor: theme.colors.gray[400],
  },
  dark: {
    color: theme.colors.whiteAlpha[900],
    bg: theme.colors.gray[800],
    borderColor: theme.colors.whiteAlpha[300],
    placeholderColor: theme.colors.whiteAlpha[400],
  },
});
```

And then pass it to the CSSReset component:

```jsx live=false
<ThemeProvider>
  <ColorModeProvider>
    <CSSReset config={customConfig} />
    <Component />
  </ColorModeProvider>
</ThemeProvider>
```

## Props

| Name     | Type     | Default       | Description           |
| -------- | -------- | ------------- | --------------------- |
| `config` | `object` | defaultConfig | Configuration object. |
