# Icon

An icon is a graphical representation of meaning. Icons are used to represent
common actions and states.

> Chakra UI can be extended to use any icon library of your choice.

## Installation

```bash
npm install @chakra-ui/core
```

then import Icon

```js
import { Icon } from "@chakra-ui/core";
```

## Icons are not buttons

An icon is only a `svg` element and should not have `onClick` handlers. If you
need a clickable icon, use [IconButton](/iconbutton) instead.

## Props

| Name    | Type                  | Default        | Description                                   |
| ------- | --------------------- | -------------- | --------------------------------------------- |
| `size`  | `string`              |                | The size of the icon.                         |
| `name`  | `string`              |                | The name of the icon.                         |
| `color` | `string`              |                | The color of the icon.                        |
| `role`  | `presentation`, `img` | `presentation` | The role of the icon. `presentation` or `img` |
