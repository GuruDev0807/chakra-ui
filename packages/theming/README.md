## Theming Framework for Chakra UI

Every theme must have the following structure

- theme
  - foundation
    - breakpoints
    - color
    - typography
    - sizing
    - spacing
    - radius
    - motion
    - shadows
    - index.ts
  - components
    - Button
    - Badge
    - Accordion
    - Checkbox
    - Radio
    - ...
    - index.ts
  - styles
    - index.ts
  - index.ts
